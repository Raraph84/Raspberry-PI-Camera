# Raspberry PI Camera

Ce programme permet de créer un serveur web contenant stream.mjpg, le stream du flux de la caméra du Raspberry PI.
Fonctione sous Raspberry PI OS Buster

## Installation

N'oubliez pas d'installer les librairies :
```
npm install
```
